INTRODUCCIÓN A HTML

Comentado el 5 de mayo de 2020

Con las herramientas de esta sección se puede escribir un párrafo pues se puede dar enter, se puede poner cursiva, negrillas, se puede hacer una lista de elementos sin orden o con orden y se puede insertar una imagen y darle el ancho deseado.

Etiqueta html Significado
br enter
em letra cursiva
strong negrillas
ol lista ordenada
ul lista sin orden
li elemeto de la lista
img src="" alt="" width="" inserta una imagen con un nombre de referencia y ancho dado

INTRODUCCIÓN A CSS

Comentado el 5 de mayo de 2020

Para dar estilo al documento html hay que selecionar cada elemento y entonces designar atribulos dentro de ellos. Muchos elementos html pueden tener una misma clase pero solamente un elemento html puede tener un id.

Selector CSS Significado Ejemplo de etiqueta en html necesaria
body{} selecciona cuerpo
h1{} selecciona título de tamaño h1
h2{} selecciona título tamaño h2
p{} selecciona párrafo
#tema{} selector de id h2 id="tema"
.roja selector de clase strong class="roja"

Atributo CSS Significado
color: rgb(0,0,0); color fuente blanco
background: rgb(256,256,256); color de fondo negro
width: 300px ancho de 300 píxeles

Más etiquetas HTML

Comentado el 8 de mayo de 2020

Con estas etiquetas puedes crear hipervínculos, también tablas con cabecera, celdas en la cabecera, filas y celdas en las filas.

Etiqueta html Significado
a href="https://latam.aleks.com/" etiqueta hipervínculo, href dice a donde te dirige
table crea una tabla
thead crea cabecera de la tabla
tr crea una fila de la tabla
th crea una celda en la fila de cabecera de la tabla
td crea una celda en una fila de la tabla

Propiedades de texto CSS

Comentado el 8 de mayo de 2020

Con estos atributos de CSS se puede dar estilo a la letra (fuente) y al contenedor del texto.

Atributo CSS Significado
font-family:cursive; tipo de letra, puede ser: cursive, monospace, fantasy, sans-serif
font-size:3px; tamaño de letra
font-family: "Helvetica", sans-serif; cascada, se escoje la primera, si no existe, se escoje la segunda
font-weight: bold; letra en negrilla
font-weight: 700; el mismo grosor de bold, se puede usar valores de 100 a 900
font-style: italic; estilo de letra italic y oblique
text-decoration: underline; subraya el texto
text-align: center; alinea el texto left, center, right, justify
text-decoration: none; quita el subrayado, se usa también overline, line-through, underline, underline overline
background-color: yellow; color de fondo
line-height: 40px; altura del contenedor

Herramientas de desarrollo web

Comentado el 8 de mayo de 2020

Usar la herramienta de inspeccionar HTML y CSS haciendo click derecho sobre cualquier parte del navegador para ver el código.

Usar editores en línea como Codepen. Usar editores de escritorio como Visual Studio Code o Sublime Text. Usar editores de línea de comando como Vim, ya vienen instalados en la computadora, son la consola.

Se puede hospedar páginas web en servidores (computadoras) de pago, en tu propia computadora (nadie de fuera podrá visitarlas) o en Github que es un servidor gratuito.
