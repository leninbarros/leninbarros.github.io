Catálogo de viaje

Comentado el 5 de mayo de 2020

Diseñé un catálogo de viaje, con fotografías tomadas de Wikipedia, en HTML. Le puse estilo con CSS.
Aprendí etiquetas de HTML, selectores de CSS y atributos de CSS. A continuación lo aprendido:

Etiqueta html Significado
em punto aparte
strong letras negrillas
ol lista ordenada
ul lista sin orden
li elemeto de la lista
img src="" alt="" width="" inserta una imagen con un nombre de referencia y ancho dado
.nombre-de-clase selecciona clase

Selector CSS Significado
body{} selecciona cuerpo
h1{} selecciona título de tamaño h1
h2{} selecciona título tamaño h2
p{} selecciona párrafo

Atributo CSS Significado
color: rgb() cambia color de fuente
background-color: rgb() Cambia el color del fondo
width: 300 ancho de 300

Blog de Lenin HTML-CSS-JS

Comentado el 9 de mayo de 2020

Diseñé un recetario y aprendí la escritura de id´s y llamarlos en CSS.
Aprendí etiquetas de HTML, selectores de CSS y atributos de CSS. A continuación lo aprendido:

Etiqueta html Significado
id="nombre-id" designa un id
class="nombre-clase" designa una clase
a href="#nombre-id" hipervínculo que lleva al id

Selector CSS Significado
.nombre-clase selector de clase

Atributo CSS Significado
font-size:2.5em; tamaño letra
font-family: "Courier", monospace; tipo de letra (sans-serif, serif, fantasy, monospace, and cursive)
font-style: italic; estilo de letra()
line-height: 1.5em; altura de línea
text-align: center; alinear (left, center, right)
text-decoration: underline; subrayado
